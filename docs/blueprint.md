# **App Name**: TrustNet Hub

## Core Features:

- Gemini-Powered Advisor: Integrate Gemini AI model as a cybersecurity expert chatbot with online status.
- Theme System: Implement a dual-branding theme system with persistent light/dark mode preference.
- Admin Panel: Secure /admin route to manage blog posts, services, and social settings.
- Lead Generation: Contact forms with validation. Route leads to admin panel.
- Expert Blog: Category filtering, sharing and URLs
- Service Catalog: Display service with modern icons.

## Style Guidelines:

- Primary color: Electric Blue (#7DF9FF) to reflect a cutting-edge cyber aesthetic.
- Background color: Very dark desaturated blue-gray (#121822) for a dark scheme.
- Accent color: Cyan (#00FFFF) to complement and highlight key interactive elements.
- Body and headline font: 'Inter', sans-serif, for a modern and neutral look.
- Lucide-react icons throughout the site.
- Dynamic grids showcase security expertise. The forms will be displayed in an interactive and well-validated style.
- Smooth animations during loading and when a user interacts with TrustNetBot.
